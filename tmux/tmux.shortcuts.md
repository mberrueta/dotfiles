# TMUX

?     shows a list of all commands (q close)
:     enter a tmux command
d     detach from session

## Working with Windows

c     creates a new window
&     kill window
,     rename current window
f     find window
p/n   switch to previous/next window
w     list windows (and then select with arrow keys)

## Working with Panes

%/-     split window vertically/horizontally
arrows  go to pane
q       show pane numbers
x       kill pane
pref {  Move the current pane left
pref }  Move the current pane right
pref z  toggle pane zoom
